# AnsibleScripts
=================

Мой набор Ansible-скриптов для работы с СУБД Oracle

system_prepare_for_oracle - подготовка OS  Linux для установки oracle database

system_install_java_8 - установка java 8

oracle_install_11_2_0_4  - установка ПО Oracle 11.2.0.4

system_sshd_oracle_on и system_sshd_oracle_off - включение и отключения прямого доступа для пользователя oracle

oracle_install_12_2 и oracle_install_12_2_patch - установка ПО oracle версии 12.2
